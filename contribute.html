The archive is built from community contributions. If you have photographs of Formula One cars from the tobacco livery era — either with branding visible or removed — this is how to add them.

How It Works
Three Steps
1
Name Your File Correctly

Every photograph must follow an exact filename convention so the archive can parse year, constructor, brand, and livery type automatically. Use the filename builder below to generate the correct name.

Supported formats: .jpg  ·  .jpeg  ·  .png  ·  .webp

2
Fork the Repository

Fork the Sans Tabac repository on GitHub, drop your named image file into the photos/ directory, and commit it.

You can add multiple files in a single commit. The manifest is regenerated automatically on every push via GitHub Actions — you do not need to edit data/photos.json by hand.

3
Open a Pull Request

Open a pull request against the main branch. Include a brief note on the source of the photograph (race, event, photographer credit if known). The pull request will be reviewed and merged if the photo and filename are valid.

Filename Reference
Naming Convention
YEAR_Constructor_Brand_[full|sans]_optional-description.jpg
Use hyphens within multi-word constructor/brand names and descriptions. No spaces anywhere.
Part Format Examples
YEAR 4-digit season year 1988, 1994, 2005 Required
Constructor Team name; hyphens for spaces McLaren, BAR-Honda, Lotus Required
Brand Tobacco brand; hyphens for spaces Marlboro, Lucky-Strike, John-Player-Special Required
full / sans Whether tobacco branding is visible full = ads shown  ·  sans = ads removed Required
description Optional context; hyphens for spaces monaco-pitlane, qualifying-session, podium Optional
Examples
◎ 1988_McLaren_Marlboro_full_monaco-pitlane.jpg
◎ 1994_Williams_Rothmans_sans_france-qualifying.jpg
◎ 2005_Ferrari_Marlboro_sans_european-gp-scrutineering.jpg
◎ 1984_Lotus_John-Player-Special_full.jpg
◎ 1999_BAR-Honda_Lucky-Strike_full_pit-stop.jpg

Before You Contribute
Copyright & Rights

Only contribute photographs that you took yourself or that you have explicit permission to redistribute. Do not submit photographs scraped from news archives, wire services, or commercial photo agencies without verifying the licence.

By opening a pull request you confirm that you have the right to share the image under the terms of this archive. All photographs displayed on Sans Tabac remain the property of their respective creators. This site provides an index only and is not affiliated with Formula One Group, any constructor, or any tobacco company.

If you would like a photograph removed, open an issue on the GitHub repository with the filename and reason.
